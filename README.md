## 게임 컨셉

### High Concept
Modern Driller는 클래식 퍼즐 액션 게임 "미스터 드릴러"에 영감을 받아 제작된 안드로이드 게임입니다. 이 게임은 플레이어가 다양한 색상의 블록을 뚫고 내려가며, 산소 레벨을 관리하고, 다양한 장애물을 피하면서 최대한 깊이 드릴링하는 것을 목표로 합니다.

### 핵심 메카닉
- **드릴링**: 플레이어는 다채로운 블록을 뚫고 내려가야 합니다. 각 블록은 고유의 색상들을 가지고 있습니다.
- **산소 관리**: 산소 레벨을 유지하는 것이 중요합니다. 산소 레벨이 다 떨어지기 전에 산소 탱크를 수집해야 합니다.
- **장애물**: 장애물이 플레이어의 진행을 방해합니다. 이를 피하거나 극복하는 전략이 필요합니다.
- **업그레이드 시스템**: 드릴, 산소 탱크, 캐릭터 능력치 등을 업그레이드할 수 있는 시스템입니다.

## 개발 범위

- **캐릭터**: 캐릭터 1명
- **블록 타입**: 4가지 기본 블록 
- **장애물**: 장애물 블록
- **UI/UX**: 메인 메뉴, 게임 플레이 화면
- **사운드**: 배경음악 1곡, 효과음 3종류

## 개발 일정

### 주차별 상세 개발 일정

- **1주차 (4월 4일 ~ 4월 10일)**: 필요한 리소스 확보 및 게임 플레이 화면 구성. 
- **2주차 (4월 11일 ~ 4월 17일)**: 기본 게임 개발 시작, 캐릭터 및 블록 디자인
- **3주차 (4월 18일 ~ 4월 24일)**: 플레이어 움직임 구현, 블록 생성 로직 구현.
- **4주차 (4월 25일 ~ 5월 1일)**: 블록이 사라지고 떨어지는 로직 구현.
- **5주차 (5월 2일 ~ 5월 8일)**: 게임플레이 화면 UI 구현.
- **6주차 (5월 9일 ~ 5월 15일)**: 장애물 생성, 아이템 로직 구현.
- **7주차 (5월 16일 ~ 5월 22일)**: 사운드 넣기.
- **8주차 (5월 23일 ~ 5월 29일)**: 미흡했던 작업들 보수.
- **9주차 (5월 30일 ~ 6월 5일)**: 최종 검수.
